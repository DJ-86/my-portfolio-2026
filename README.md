# Dave Jones — Portfolio

Personal portfolio site for Dave Jones, frontend developer based in London, UK.

**Live site:** [davejonesuk.dev](https://davejonesuk.dev)

---

## About

A clean, dark-themed portfolio built with vanilla HTML, CSS, and JavaScript — no frameworks, no build tools, no dependencies beyond Google Fonts. The goal was to keep it straightforward and easy to maintain.

---

## Structure

```
portfolio/
├── index.html      # Markup and content
├── style.css       # All styles and responsive layout
├── main.js         # Scroll behaviour and nav highlighting
└── README.md
```

---

## Features

- Fixed nav with active section highlighting on scroll
- Animated hero on page load (CSS keyframes)
- Skills grid with a "currently learning" category
- Project cards linking to live demos
- About section with timeline
- Fully responsive down to mobile
- No build step — open `index.html` directly in a browser

## Running locally

No setup required. Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/DJ-86/portfolio.git
cd portfolio
open index.html   # macOS
# or just drag index.html into a browser window
```
